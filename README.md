## 🚨 Tutorial

This repository contains the code corresponding to the project of a free platform that collects and filters nanosatellite's data to the public.

## <a name="introduction">🤖 Introduction</a>

NanoSatTracker App serves as a mobile platform with the primary goal of making data generated by nanosatellites accessible to the general public. It achieves this by offering a website and mobile application equipped with interactive dashboards. These dashboards play a vital role in presenting and visualizing the key telemetry and information derived from the satellite's mission interface. In essence, NanoSatTracker enables anyone to conveniently access and understand the valuable data produced by nanosatellites. It is responsible for creating a dashboard showing the communication between the earth (a ground station) and a satellite, that are divided in two sub-modules: Beacon and downlink/uplink.

## <a name="features">🔋 Features</a>


👉 **Robust Authentication & Authorization System**: Secure email login safeguards user accounts.

👉 **Spacelab main page**: A simple page focused on explaining the Spacelab's space missions and activities

👉 **FloripaSat-1 main page**: A simple page focused on explaining the FloripaSat-1 mission.

👉 **Downlink**: Information about all the data collected by the FloripaSat-1 mission.

👉 **Battery**: A page focused on displaying data about the FloripaSat-1's Battery system.

👉 **Solar Panels**: A page focused on displaying data about the FloripaSat-1's Solar Panels system.

👉 **Locations**: The Locations page displays information about the ground stations around the world that have made contact with the FloripaSat-1.

👉 **Ham Radio**: The Ham Radio page displays information and gives credit to the Ham Radio that helped collect the mission's data.

👉 **OBDH- data**: This page is not ready to be launched, but will display more detailed data about the On-Board Data Handling board in the FloripaSat-1 mission.

👉 **Responsiveness**: Smooth performance and adaptability across various devices and screen sizes for a consistent user experience.

## <a name="quick-start">🤸 Quick Start</a>

Follow these steps to set up the project locally on your machine.

**Prerequisites**

Make sure you have the following installed on your machine:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/) (Node Package Manager)

**Cloning the Repository**

```bash
git clone [https://github.com/adrianhajdin/aora.git](https://github.com/kiqsmg/NanoSatTracker-App.git)
cd nanosattracker_application
```
**Installation**

Install the project dependencies using npm:

```bash
npm install
```

**Running the Project**

```bash
npm start
```

**Expo Go**

Download the [Expo Go](https://expo.dev/go) app onto your device, then use it to scan the QR code from Terminal and run.
